# MindMeshLab — Contact Landing

A small multilingual (DE · EN · AR) landing page linked from a QR code.
Visitors can choose to open WhatsApp chat or visit the main website.

## Files
- `index.html` — the landing page
- `mindmeshlab_qr.png` — QR code pointing to the deployed site

## Deployment
Deployed as a Render Static Site. Publish directory: `.`
